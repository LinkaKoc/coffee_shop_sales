# data_portfolio-coffee_shop_sales
Firma chce zjistit:  
- které produkty vydělávají nejvíc,
- kdy je nejvyšší provoz,
- které pobočky mají nejslabší výkon.  

Udělej:
- SQL analýzu,
- 1 dashboard v Power BI,
- krátké business doporučení. 

KPI:
- daily sales
- average basket size
- top 5 products
- sales by hour
- sales by location

Data z https://www.kaggle.com/datasets/navjotkaushal/coffee-sales-dataset
